## Desafio: Classificador de Nível de Herói
Este projeto foi desenvolvido como parte de um desafio prático para consolidar conceitos fundamentais de programação, focando em variáveis, operadores e estruturas de decisão.

## Estrutura Utilizada: Switch(true)
Neste programa, utilizei a estrutura switch com uma abordagem lógica avançada: o switch (true).

## O que o código faz ?
O código recebe a quantidade de experiência (XP) de um herói e, através de uma lógica de faixas de valores, classifica-o em níveis que vão desde:

Nível,XP
Ferro,Menor que 1.000
Bronze,Entre 1.001 e 2.000
Prata Ouro,Entre 2.001 e 5.000
Platina Diamante,Entre 5.001 e 8.000
Ascendente,Entre 8.001 e 9.000
Imortal,Entre 9.001 e 10.000
Radiante,Maior ou igual a 10.001

## Execute o programa digitando:
node desafio-filipao.js
